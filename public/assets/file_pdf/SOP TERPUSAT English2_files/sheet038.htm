  a. Jika masuk Pulau Jawa, Bali, Kalimantan, Lampung, Babel, Sumbar, NTB :
- catat laporan pada APKT gangguan
- tetangga tidak turut padam
- input permasalahan (koresponden c) klik IML dilayani listriqu
b. Jika diuar Pulau Jawa, Bali, Kalimantan, Lampung, Babel, Sumbar, NTB :
- arahkan pelangan menghubungi CC ESDM di No 021-136 atau dapat mengakses melalui website DJK : slodjk.esdm.go.id/instalatir/terdaftar
- catat laporan pada aplikasi (issued closed di CC 123) koresponden d
    
15 Memberikan penegasan informasi atas kejelasan : "apakah informasi yang saya berikan sudah cukup jelas?"

  jika menanyakan estimasi penanganan maka sampaikan hal tsb akan disampaikan oleh petugas call center listriqu
