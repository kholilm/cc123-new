GANGGUAN – TIDAK DAPAT INPUT TOKEN
SOP 2 FEBRUARI 2021
CASE PROBING ISSUE TYPE WORK ORDER
TIDAK DAPAT INPUT TOKEN  1 "PLN 123, Selamat (.....) dengan (nama CSO) bisa dibantu?" INPUTKAN PERMASALAHAN KE APKT GANGGUAN BAIK KONDISI LISTRIK PADAM/ MENYALA

* Tipe Permasalahan untuk Gangguan :
- jika tetangga tidak turut padam: 1 RUMAH PADAM
- jika tidak tahu tetangga padam atau tidak: 1 RUMAH PADAM

* Tipe Permasalahan untuk Keluhan :
Tipe : APP - Alat Ukur Prabayar tidak Berfungsi (token tidak dapat diinputkan kondisi kwh belum dilakukan upgrade oleh petugas)

Tipe : type : tagihan listrik dan token
subtype : konversi token
jika menyala, tdk tahu sisa kwh dan sisa kwh >10 (jika kwh sudah di upgrade namun token lama tidak bisa diinputkan maka WO konversi token)		 PELANGGAN TIDAK BISA INPUT TOKEN SETELAH UPDATE VKRN (PTL PADAM/MENYALA)
2 Menanyakan nama penelepon/menyapa nama pelanggan diawal
3 Menanyakan perihal kelistrikan
4 Menanyakan apakah pernah melapor sebelumnya?
(jika pernah melapor maka lihat SOP lapor ulang)
5 Menanyakan display kwh tertera apa
6 Menanyakan no meter
  Lakukan pengecekan data sesuai idpel pada aplikasi (AP2T)
Menu : Info DIL - kwh (VKRN, KRN)
a. Jika tertera 41 atau 42 : kwh meter belum di update (maka lanjutkan sop seperti prabayar 1 rumah padam)
b. Jika tertera 43 : kwh meter sudah di update
- tanyakan apakah sudah menginput token dengan nomor sesuai
- tanyakan apakah di lokasi sudah dilakukan pembaruan pada kwh meter pelanggan
- infokan bahwa permasalahan akan disampaikan ke unit lanjutkan probis selanjutnya
7 Menanyakan kondisi listrik padam/menyala :
a. Jika padam tanyakan posisi mcb MCB di kwh meter dan instalasi dalam bangunan dalam posisi ON/OFF
b. Jika menyala menanyakan sisa kwh :
Note :
sisa kwh <10 kwh atau ptl padam : input Gangguan
sisa kwh >10 kwh atau tidak mengetahui sisa stroom : input Keluhan
9 Ucapkan magic words (disesuaikan dengan case)
10 Menanyakan Alamat lengkap
(jika pelanggan sudah menginfokan no meter/idpel maka bisa langsung verifikasi alamat sesuai DIL (jalan besarnya) dan menambahkan alamat yang kurang lengkap)
11 "Apakah bapak/ibu dapat dihubungi kembali dengan nomor yang telpon yang digunakan saat ini?"
12 Laporan Bapak / Ibu kami sampaikan ke unit terkait untuk segera ditindaklanjuti
13 Menyampaikan kejelasan informasi "apakah informasi yang kami sampiakna sudah cukup jelas?"
13 Mohon dicatat nomor lapor sebagai nomor bukti pelaporan kepada kami.
14 Ada lagi yang bisa dibantu? (disesuaikan dengan kondisi KLB/tdk)
15 Jika pelanggan menanyakan estimasi tindaklanjutnya maka disampaikan 
“... estimasi waktu penyelesaiannya ± 3 (tiga) jam dari waktu lapor"
16 "Untuk mewujudkan PLN TERBAIK, mohon tidak memberikan tambahan biaya kepada petugas kami.
Agar semua makin mudah, ayo  download aplikasi New PLN Mobile di Play Store atau Apps Store,  Terimakasih telah menghubungi PLN 123, selamat siang {Bp/Ibu... Nama Pelanggan...}" (disesuaikan dengan kondisi KLB atau tidak)
